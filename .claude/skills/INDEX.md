## Hur man använder skills

Skills aktiveras genom att referera till dem i en konversation med Claude Code.
Varje skill-fil innehåller en YAML-frontmatter med `name`, `description`, och triggers.

```
/skill marketing/seo-audit        # Kör SEO-revision
/skill business-growth/revenue-operations  # RevOps-analys
```

Eller inkludera skill-filen i kontexten:
```bash
cat .claude/skills/marketing/cold-email.md | claude -p "Skriv kall e-post till webbyråer"
```
